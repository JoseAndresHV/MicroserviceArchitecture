## Docker commands
``` console 
docker pull postgres
```

``` console
docker run --name postgres-database --network micro -v c:/docker/micro/postgres:/var/lib/postgresql/data -e POSTGRES_PASSWORD=Passw0rd -d -p 5434:5432 postgres
```

## Database scripts
``` sql
CREATE DATABASE db_account;

CREATE TABLE Transaction(
  Id INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  Amount DECIMAL,
  Type VARCHAR(30),
  CreationDate VARCHAR(50),
  AccountId INTEGER
);
```
